# 🚨 Incident Management System (Python CLI)

A simple **Incident Management System** built using pure Python.  
This project simulates the real-world incident lifecycle including creation, assignment, tracking, SLA monitoring, and reporting — all without using a database.

---

## 📌 Features

- 👤 User Management (Admin, Agent, User)
- 📝 Incident Creation
- 📊 Incident Listing
- 🔄 Incident Assignment
- 📌 Status Tracking (Open → In Progress → Resolved → Closed)
- ⏱️ SLA Monitoring
- 📈 Basic Reporting
- 💾 In-memory storage (no database required)

---

## 🛠️ Technologies Used

- Python 3
- Built-in modules:
  - `uuid`
  - `datetime`
